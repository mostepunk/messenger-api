# Healthcheck Module
Модуль проверки состояния коннектов.

## Описание модуля
```
 healthcheck_module
 ├── api
 │   └── routes
 │       └── healthcheck.py
 ├── db
 │   ├── cruds
 │   │   └── health_crud.py
 │   └── models
 ├── schemas
 ├── services
 │   └── health_service.py
 └── tests
     ├── conftest.py
     ├── test_api.py
     └── test_service.py
```
